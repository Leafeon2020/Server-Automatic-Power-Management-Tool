# サーバー電源管理Botのテストブランチ
本ブランチは**開発途中の物**となります。そのため、動かなかったりエラー吐いて止まったりします。つまり**正常に動きません**。(テスト済)

## 本ブランチの用途
自分でコーディングしていて突破口が見えないので一旦仮公開する形で意見を求める形となっています。
